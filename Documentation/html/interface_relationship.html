Detailed Description

A subclass of Component containing the data related to a relationship between two variables of a causal loop diagram. ex. In a diagram on Childhood obesity, there may exist a relationship between the "Fast Food Consumption" variable and the "Weight Gain" variable. This class represents that relationship.
